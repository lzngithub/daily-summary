# 运算符

## 分类

粗略的分

- 一元运算符
- 二元运算符
- 三元运算符

细致的分

- 赋值 = += -= \*= /= %=
- 比较: < <= > >= == === !==
- 算数: + - \* / % \*\*
- 位: & | ^ ~ << >> >>>
- 逻辑: && || !
- 字符串: + +=
- 条件运算符: ?:
- 逗号: ,
- 一元操作符: delete typeof void
- 关系: in instanceof
