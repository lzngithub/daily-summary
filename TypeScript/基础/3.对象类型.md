# 对象类型

可以通过接口来定义对象的类型

## 接口

接口是行为的抽象，具体如何行动是通过类去实现的。

例子：

```ts
interface Person {
  name: string;
  age: number;
}

let her: Person = {
  name: "li",
  age: 18,
};
```

上面是简单的例子，可以对接口定义的属性进行一些约束

可选属性，用 ? 操作符来定义

```ts
interface Person {
  name: string;
  age?: number;
}

let her: Person = {
  name: "li",
};
```

任意属性，用 [] 符号来定义

```ts
interface Person {
  name: string;
  [otherName: string]: any;
}

let her: Person = {
  name: "li",
  height: 18,
};
```

只读属性，关键字 readonly 定义

```ts
interface Person {
  readonly id: number;
  name: string;
  [otherName: string]: any;
}

let her: Person = {
  id: 123456,
  name: "li",
  height: 18,
};

her.id = 123;
```

上面修改 id 属性会报错
