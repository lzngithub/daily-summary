## 权重

| 选择器                         | 权重       | 权重的等级 |
| :----------------------------- | :--------- | :--------: |
| ！important                    | 无穷大     |     0      |
| style 样式                     | 1，0，0，0 |     1      |
| id 选择器                      | 0，1，0，0 |     2      |
| 类、伪类、结构伪类、属性选择器 | 0，0，1，0 |     3      |
| 标签、伪元素选择器             | 0，0，0，1 |     4      |
| 通配符选择器                   | 0，0，0，0 |     5      |
| 继承的样式                     | 没有权重   |     6      |
